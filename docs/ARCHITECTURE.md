# Architecture

> To be completed after Phase 3.

## Overview

Fashion Tech Pack AI is a multi-agent system that generates structured fashion
tech packs from unstructured design inputs. The system uses a hybrid Node.js +
Python architecture with dual agent frameworks (CrewAI and LangGraph).

## System Components

- **API Gateway** (Node.js / Express / TypeScript)
- **Orchestration Service** (Python / FastAPI)
- **Agent Frameworks** (CrewAI prototype + LangGraph production)
- **Vector DB / RAG** (Supabase pgvector)
- **Cache** (Redis)
- **LLM Routing** (OpenRouter)
